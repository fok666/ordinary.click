Conditions for Training Use

  1. Release model weights publicly. The resulting model weights must be publicly released under an Open Source Initiative–approved or Creative Commons license, within 90 days of first deploying, distributing, or otherwise making the model available to any third party. “Model weights” means the learned numerical parameters of the trained model in a form sufficient to reproduce its inference behaviour.
  2. Disclose the training data source. Include, in any public model card, repository, or documentation accompanying the released weights, a statement that material from my.ordinary.click (licensed under CC BY-SA 4.0 + AI Training Openness Addendum) was used during training, along with a link to this page.

Permitted without triggering this addendum

  • Using this material solely for personal research, evaluation, or experimentation where the resulting model is never shared with any third party.
  • Using this material in training pipelines where the trained model is already covered by a pre-existing open license satisfying conditions 1–3 above.

Scope clarification
